
# To Do React App

A React app to mark your important tasks.

## Demo

https://to--do-react.herokuapp.com
  
## Deployment

To deploy this project run

```bash
  npm start
```

  
## Features

- Mark done tasks
- Delete unwanted tasks
- Pop ups to notify 
- Time and Date markings
- Simple UI

## NPM packages used
- react
- react-hot-toast
- react-icons
